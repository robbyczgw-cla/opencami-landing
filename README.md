```bash

```

## Deployment

### Vercel (Recommended)

1. Connect this repo to Vercel
2. Select "Other" framework preset
3. Deploy!

Or via CLI:

```bash
npm i -g vercel
vercel --prod
```

### Manual

Just upload `index.html` and `vercel.json` to any static host.
